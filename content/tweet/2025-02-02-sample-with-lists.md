Tweetでリストやコードブロックを使うサンプルです。

## リストの例

- 最初の項目
- 二番目の項目
- 三番目の項目
  - ネストされた項目1
  - ネストされた項目2

## ナンバーリストの例

1. 最初の項目
2. 二番目の項目
3. 三番目の項目
   1. ネストされた項目1
   2. ネストされた項目2

## コードブロックの例

```javascript
function greet(name) {
    const message = `Hello, ${name}!`;
    console.log(message);
    return message;
}

greet('World');
```

これらは通常のTweetと同じように表示されます。
