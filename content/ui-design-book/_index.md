# UI Design Book

## 目的

UIデザインに関する知識、コンポーネント構造、原則、設計思想を体系化して蓄積するためのコンテンツ群です。個別のTipsではなく、**1テーマ＝1ファイル**の百科事典スタイルでまとめています。

## 構成

UI Design Bookは **Components / Patterns / Principles** の3階層で構成します。

### Components（UIの最小部品）

UIの基本パーツ。定義・使い方・状態・誤用例・アクセシビリティなどをまとめます。

### Patterns（複数コンポーネントの組み合わせ）

画面構造・インタラクションの定番パターン。目的、構造、デザインの鉄則、よくある失敗と対策、事例をまとめます。

### Principles（デザイン原則・思想）

コンポーネントやパターンを支える抽象的なルールや理論。原則の定義、重要性、適用シーン、判断基準、誤用例をまとめます。
