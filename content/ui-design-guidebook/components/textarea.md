# Textarea

## 1. 何のためのコンポーネントか（役割）

Textareaは、複数行のテキスト入力に使用するフォーム要素です。主な役割は：

- **長文入力**: コメント、説明、メッセージなどの長いテキスト入力
- **自由記述**: フォーマットが決まっていない自由な記述
- **マルチライン入力**: 改行を含むテキストの入力

## 2. よく使う場面

- **コメントフォーム**: ブログやSNSのコメント入力
- **問い合わせフォーム**: お問い合わせ内容の入力
- **レビュー**: 商品レビューや評価の記述
- **メッセージ**: メッセージやチャットの入力

## 3. 構造と状態

### 基本構造

```
[Label]
[Textarea]
[Helper Text / Character Count]
```

### 主要な状態

- **Default**: 初期状態
- **Focus**: フォーカス時
- **Filled**: 値が入力された状態
- **Error**: バリデーションエラー時
- **Disabled**: 無効状態

## 4. デザインのポイント

- **適切なサイズ**: 入力内容に応じた高さを設定（rows属性）
- **リサイズ可能**: ユーザーがサイズを調整できるように（resize）
- **文字数カウント**: 長文入力時は文字数カウントを表示

## 5. アクセシビリティ

- **ラベル**: 明確なラベルを提供
- **ARIA属性**: `aria-describedby`でヘルプテキストと関連付け
- **キーボード操作**: Tabキーでフォーカス可能

## 6. 誤用・アンチパターン

- **単行入力に使用**: 1行の入力にはInput Fieldを使用
- **サイズが固定**: リサイズ可能にするか、適切なサイズを設定

## 7. 関連するUIパターン

- **Form**: フォーム内の入力要素として使用
- **Validation**: バリデーション機能と組み合わせ
