# SP版デザイン仕様書

## 共通ヘッダー構造

### ヘッダー上部（全ページ共通）
```
┌─────────────────────────────────┐
│ ☰  ezeroms.com            🔖   │
└─────────────────────────────────┘
```

- **左上**: ハンバーガーメニュー（☰）
  - 機能: 全体の目次へ（画面下部からドロワー形式で表示）
  - 実装: 既存のハンバーガーメニュー機能をドロワー形式に変更

- **中央**: サイト名（ezeroms.com）
  - スタイル: イタリック体、太字

- **右上**: タグアイコン（🏷️）
  - 機能: タグリストがあるページは、ここに導線を置く
  - 対象ページ: Tweet, Diary, Column, Work, The shoulders of Giants
  - 表示方法: 画面下部からドロワー形式で表示

## ページタイプ別の機能

### 1. Tweet, Diary, Work ページ

#### ヘッダー構造
```
┌─────────────────────────────────┐
│ ☰  ezeroms.com            🏷️   │
├─────────────────────────────────┤
│ ‹  2025  ›  J F M A M J J A S O N D │
└─────────────────────────────────┘
```

#### 機能説明
- **ヘッダー下部**: 年月選択UI
  - 年選択: 左右矢印（‹ ›）で年を変更、中央に年を表示（例: 2025）
  - 月選択: 月の略称ボタン（J, F, M, A, M, J, J, A, S, O, N, D）を横スクロール可能な形式で表示
  - アクティブな月は強調表示

- **タグアイコン**: タグリストへの導線
  - クリックでタグリストを画面下部からドロワー形式で表示

#### 実装状況
- ✅ 年月選択UI: 既に実装済み（`tweet-header-nav`）
- ⚠️ タグアイコン: 未実装（要追加）

## 実装が必要な機能

### 1. 全体の目次（ハンバーガーメニュー）のドロワー化
- **対象**: 全ページ共通
- **機能**: ハンバーガーメニューをクリックで、画面下部からドロワー形式で全体の目次（サイドバーメニュー）を表示
- **実装場所**: `layouts/_default/baseof.html` のハンバーガーメニュー機能を修正

### 2. タグアイコンの実装
- **対象ページ**: Tweet, Diary, Column, Work, The shoulders of Giants
- **機能**: タグアイコンをクリックで、画面下部からドロワー形式でタグリストを表示
- **実装場所**: `layouts/_default/baseof.html` の `layout-header` 内

### 3. Chronicle ページの設定アイコンとドロワー
- **対象ページ**: Chronicle
- **機能**: ヘッダー右側に設定アイコン（⚙️）を表示し、クリックで画面下部からドロワー形式で設定系UIを表示
- **表示内容**: PC版で左側に表示している設定系UI（生まれ年、表示期間、カテゴリ、テーマ史）を全てドロワーに含める
- **実装場所**: `layouts/_default/baseof.html` の `layout-header` 内、`layouts/partials/chronicle-toc.html` をドロワー用に調整

### 4. Column ページのページ内メニュー
- **機能**: ヘッダー下部のハンバーガーメニューをクリックでカテゴリー選択UIを下から表示（ドロワー形式）
- **表示内容**: カテゴリー一覧（すべて、本・雑誌、デザイン・クリエイティブ など）
- **実装場所**: `layouts/column/list.html` の `section_header` を修正

### 5. About ページのページ内メニュー
- **機能**: ヘッダー下部のハンバーガーメニューをクリックで下層ページ選択UIを下から表示（ドロワー形式）
- **表示内容**: 下層ページ一覧（プロフィール、このサイトについて、メディア掲載、お問い合わせ）
- **実装場所**: `layouts/partials/about-toc.html` をドロワー用に調整

### 6. UI Design Guidebook のページ内メニュー
- **機能**: ヘッダー下部のハンバーガーメニューをクリックで目次を下から表示（ドロワー形式）
- **表示内容**: カテゴリ（Components, Patterns, Principles）とページタイトル
- **実装場所**: `layouts/ui-design-guidebook/single.html` または `layouts/_default/baseof.html`

## スタイルガイドライン

### ヘッダー上部
- 高さ: 56px
- 背景: 白
- ボーダー: 下部に1pxのボーダー（rgba(0, 0, 0, 0.08)）

### ヘッダー下部（年月選択、ページ内メニューボタン）
- 高さ: 自動（コンテンツに応じて）
- 背景: 白
- ボーダー: 下部に1pxのボーダー（rgba(0, 0, 0, 0.08)）
- 年月選択UI（Tweet, Diary, Work）: 横スクロール可能（`overflow-x: auto`）
- ページ内メニューボタン:
  - Column: フィルターアイコン（⚙️）とページ名を表示
  - About: リストアイコン（☷）とページ名を表示
  - UI Design Guidebook: リストアイコン（☷）とページパンくずを表示

### タグアイコン
- サイズ: 24px × 24px
- 位置: ヘッダー右上
- スタイル: アウトラインアイコン（タグのアイコン）
- 対象ページ: Tweet, Diary, Column, Work, The shoulders of Giants

### 設定アイコン
- サイズ: 24px × 24px
- 位置: ヘッダー右上
- スタイル: アウトラインアイコン（設定/歯車のアイコン）
- 対象ページ: Chronicle

### フィルターアイコン（ページ内メニュー）
- サイズ: 24px × 24px
- 位置: ヘッダー下部（左側）
- スタイル: アウトラインアイコン（フィルター/歯車のアイコン）
- 対象ページ: Column

### リストアイコン（ページ内メニュー）
- サイズ: 24px × 24px
- 位置: ヘッダー下部（左側）
- スタイル: アウトラインアイコン（縦3本線のリストアイコン）
- 対象ページ: About, UI Design Guidebook

### ドロワー（全体の目次、タグリスト、設定系UI、ページ内メニュー）
- 表示方法: 画面下部からスライドアップ
- 背景: 白
- 高さ: 最大80vh（画面の80%まで）
- オーバーレイ: 半透明の黒背景（クリックで閉じる）
- アニメーション: スムーズなスライドアップ/ダウン
- スクロール: コンテンツが多い場合は縦スクロール可能
- ページ内メニュー（Column, About, UI Design Guidebook）: カテゴリー/下層ページ/目次をドロワー内に表示

### ハンバーガーメニューアイコン（全体の目次）
- サイズ: 24px × 24px
- 位置: ヘッダー左上
- スタイル: 横3本線（☰）
- 機能: 全体の目次（サイドバーメニュー）をドロワーで表示

### アイコンの差別化
- **全体のメニュー（左上）**: ハンバーガーアイコン（☰） - 横3本線
- **Column ページ内メニュー**: フィルターアイコン（⚙️）
- **About / UI Design Guidebook ページ内メニュー**: リストアイコン（☷） - 縦3本線
